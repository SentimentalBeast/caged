# caged
html app
CAGED Guitar Trainer — All 5 Positions
Free browser-based guitar practice tool. No installation required — 
just open the HTML file in Chrome, Firefox, or Safari.

✅ All 5 CAGED positions visualized simultaneously on a 21-fret neck
✅ 58 chord progressions (Pop, Fusion, Jazz/Blues, Modal and more)
✅ Synchronized drum machine (Simple 4/4, Bossa Nova, Shuffle)
✅ Compatible scales panel with fretboard visualization
✅ Scale Ascend/Descend playback for ear training
✅ Transpose to any of 12 keys instantly
✅ Works 100% offline — single HTML file
