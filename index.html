📖 How to Use — CAGED Guitar Trainer Click to expand
CAGED Guitar Trainer is a browser-based practice tool that visualises all five CAGED chord shapes across a 21-fret guitar neck simultaneously, synced to a chord progression backing track and drum machine. No installation — open the HTML file in any modern browser and start playing.
🎸 Interactive Fretboard
  • All 5 CAGED positions shown at once, colour-coded by shape ■ C ■ A ■ G ■ E ■ D
  • Click any shape box on the fretboard to focus that position
  • ⊕ +12F button reveals octave positions (frets 12–21)
  • Scale overlay colours: ● Root   ● 3rd   ● 5th   ● ♭7   ● Other
🎵 Chord Progressions (58 total)
  • 6 style groups: Pop/Rock · Fusion/Jazz-Rock · Jazz/Blues · J-Pop · Modal · Complete Songs
  • Set Key, Bars/Chord (1 / 4 / 8), and Loop / Once
  • Click any chord pill in the strip to open the Compatible Scales panel
  • Click a scale chip → secondary fretboard shows that scale
🥁 Drum Machine
  • Three patterns: Simple 4/4 · Bossa Nova · Shuffle
  • Select a pattern, then click ▶ Start Drums
  • Starting drums while backing track plays → beat-accurate sync (snaps to next 16th note)
  • BPM is shared with the backing track
🎼 Backing Track Player
  • Click ▶ Play Backing Track — chords highlight on fretboard in real time
  • Bars/Chord: how many bars each chord holds (1 = fast changes, 8 = slow practice)
  • 🔁 Loop keeps cycling; once-through stops after one pass
🔬 Compatible Scales & Scale Fretboard
  • Click any chord pill → panel shows scales that work over that chord
  • Click a scale chip → scale appears on the lower fretboard canvas
  • ▶ Scale Ascend / Descend plays the scale ascending then descending for ear training
⚡ Quick Start
  1. Choose a Key and Chord Progression
  2. Set BPM, choose a drum pattern, click ▶ Start Drums
  3. Click ▶ Play Backing Track
  4. Watch the fretboard — practice the highlighted CAGED shape
  5. Click a chord pill → pick a scale → improvise!
💡 Tips: Set Bars/Chord to 4 or 8 for slower practice · Use Focus Shape to isolate one CAGED position · Try Modal/Exotic progressions for advanced colour · Transpose any progression by changing Key
